TODO List (Vue版)
==================
这是一个简洁的清单应用，具有添加、删除、更新、本地存储功能。

## 运行项目
只需要在chrome浏览器下直接运行index.html文件，无需安装其他依赖<br/>

## 添加功能
• 在输入框中输入想添加的内容，点击submit或按回车后将todo项添加到下方列表的最后，并清空入框<br/>

## 删除功能
• 点击删除，可以删除这条todo<br/>

## 更新功能
• 点击更新，可以把添加的内容加入输入框下方的文本框并显示<br/>

## 存储功能
• 使用localStorage将数据保存在本地，刷新页面也不会丢失<br/>
